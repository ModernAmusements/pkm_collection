1) Regular Pokémon (Non-Rule Box)

Layout Characteristics
	•	Colored frame matching type
	•	HP top right
	•	Evolution box top left
	•	Attacks centered mid-lower
	•	Weakness/Resistance/Retreat bottom row
	•	Small flavor text box
	•	No “Rule Box”

Effect on layout: Clean, symmetrical structure. Most text space reserved for attacks.

⸻

2) Pokémon ex (modern lowercase “ex”)

Layout Changes
	•	Prominent “ex” next to name
	•	Rule Box at bottom (“When Knocked Out, opponent takes 2 Prize cards”)
	•	Higher HP
	•	Gloss/foil treatment
	•	Silver borders (SV era)

Effect on layout:
The rule box compresses the lower text area → less space for flavor text. Visual emphasis shifts to name + HP.

⸻

3) Tera Pokémon ex

Layout Changes
	•	Crystal crown icon
	•	Sparkling fractured foil pattern
	•	Sometimes altered attack box styling
	•	Tera rule clause added

Effect on layout:
More vertical stacking of rules text → attack area slightly compressed. Strong top-heavy visual weight.

⸻

4) Full Art Pokémon (ex Full Art)

Layout Changes
	•	Artwork covers entire card
	•	Text boxes float above art
	•	Heavy texture foil
	•	Minimal colored frame

Effect on layout:
Hierarchy shifts from structured frame → poster-like composition. Text readability relies on contrast overlays.

⸻

5) Illustration Rare / Special Illustration Rare

Layout Changes
	•	Border retained (usually)
	•	Large scenic artwork
	•	Minimal attack area emphasis
	•	Artistic composition prioritized

Effect on layout:
Art dominates. Text is compact and visually secondary. Strong narrative focus.

⸻

6) Gold / Hyper Rare

Layout Changes
	•	Monochrome gold palette
	•	Heavy texture
	•	Flattened color contrast
	•	Same structural layout as base card

Effect on layout:
Structure unchanged, but visual hierarchy flattens due to single-tone scheme.

⸻

7) Trainer Card Variants (Modern)
	•	Item
	•	Supporter
	•	Stadium

Layout Differences
	•	Large top header (ITEM / SUPPORTER / STADIUM)
	•	Bigger text body than Pokémon cards
	•	Supporters often get Full Art versions
	•	Stadium has centered art + large text block

Effect on layout:
Text density higher than Pokémon cards. Visual rhythm more editorial.

⸻

Key Modern Design Shifts (Scarlet & Violet Era)
	1.	Silver borders (instead of yellow)
	2.	Cleaner typography
	3.	Larger HP numbers
	4.	Reduced shadowing/gradient
	5.	More art-forward rare tiers
